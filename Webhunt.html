Opacity

the opacity aspect can be used as a standalone property or inside the HSLA value. It takes a value between 0 and 1. 1 meaning the image is solid and 0 meaning it is totally transparent. I used the opacity affect along with the hover selector to make the above image become more transparent when hovered over.

Rgb Values

an rgb value is a value made up of three numbers, up to 255 that represent the red, green and blue attributes of the item

Hex Codes

A hex code is a six digit code that references one specific colour. I used a hex code to define the background colour of this.

Hue

hue is part of a three input HSL colour value. H in HSL stands for hue. The hue is the degree on the colour wheel. 0 being red, 120 being green, 240 being blue.

Saturation

Saturation is the second input in the HSL value. it is a percentage value. )% means a shade of grey, whereas 100% is the full colour

Brightness (lightness?)

lightness/brightness is the third input for a hsl value. it is a percentage, 0 being black and 100 being white. I used the H,S,L value to define the background colour of this page.

Contrast

Contrast is the difference between the foreground and background colours. higher contrast generally means the text is easier to read. I checked my contrast for my foreground and background and found the contrast ratio to be 10.62

CSS3 HSLA

CSS3 is the latest form of the cascading style sheet language. HSLA is similar to the HSL format but allows for a fourth option, being alpha, or opacity. I used HSLA to change the opacity and colour of this
